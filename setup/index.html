Setup a Message

Create a new email

From the Campaigns page, select Create Campaign.

Name your campaign, beginning with the job number.

Select the Campaign Type: Regular

Recipients

Select Fordham Master List.

If you are using a segment that is already built select Saved or pre-built segment and select your segment.

If you are building a new segment or uploading a list, select Group or new segment.

Campaign Info

In Campaign Info fill in the Campaign name, Subject, From name, and From address.
Select Use Conversations to manage replies and Personalize the “To” Field.
In Specify merge tags put *|FNAME|* *|LNAME|*.

In the Use Conversations section select the reply notifications link and select which accounts should receive replies to this email.

Select:

  • Track opens
  • Track clicks
  • Track plain-text clicks
  • Google Analytics link tracking

Set a Title for the campaign to a unique name that will identify the traffic from this email in Google Analytics. Include the job number if available (e.g. 17-0081_Boston_Reminder).

Copy an email

Find the message you would like to copy.

Select the down arrow next to Edit and choose Replicate.
