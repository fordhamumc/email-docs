Content Wrappers

Default Content Trigger: col1

The default single-column content row.

<tr><td class="inner contents">
    {content}
    <div>&nbsp;</div>
</td></tr>

Image Row Image width: 945 | Rendered size: 630 Trigger: image

A large image that takes up the entire content area.

<tr><td class="inner contents col-img--solo">
    <img src="{image 945px wide}" width="630" alt="{alt}"/>
</td></tr>

Image Row (Full Width) Image width: 1095 | Rendered size: 730 Trigger: imagefull

A large image that takes up the entire message (without the white border).

<tr><td class="inner contents fullimage">
    <img src="{image 1095px wide}" width="730" alt="{alt}"/>
</td></tr>
