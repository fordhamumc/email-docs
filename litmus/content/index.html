Content Styles

Button Trigger: btn

The default button size.

<a class="btn" href="{url}" target="_blank"><!--[if (mso)]>&nbsp;&nbsp;&nbsp;&nbsp;<![endif]-->
    {Text}
	<!--[if (mso)]>&nbsp;&nbsp;&nbsp;&nbsp;<![endif]--></a>

Button (Large) Trigger: btn-lg

The large button is usually used in the lead content area and should only be used once in an email.

<a class="btn-lg" href="{url}" target="_blank"><!--[if (mso)]>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<![endif]-->
    {Text}
<!--[if (mso)]>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<![endif]--></a>

Button (Small) Trigger: btn-sm

The small button is used in tight spaces like the events list.

<a class="btn-sm" href="{url}" target="_blank"><!--[if (mso)]>&nbsp;&nbsp;<![endif]-->
    {Text}
<!--[if (mso)]>&nbsp;&nbsp;<![endif]--></a>

Lead Event Info Trigger: leadevent

Lead content that goes at the top of a single event invitation.

<div class="leadstory">
    <h1 class="link">{Title}</h1>
    	<h3>{Date}</h3>
    <div class="lead">
        <strong>{Location Name}</strong><br/>
        {Address}
    </div>
</div>

Lead Content Trigger: lead

Larger type that generally is used for the first paragraph or content that needs emphasis.

<div class="lead">{content}</div>

Caption Trigger: caption

The caption that goes below an image.

<div class="col-caption"><small>{caption}</small></div>
